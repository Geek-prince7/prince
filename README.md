## ğŸ¨ Customization

### Update Personal Information

1. **Contact Details** - Edit `components/Contact.tsx`
2. **About Section** - Edit `components/About.tsx`
3. **Experience** - Edit `components/Experience.tsx`
4. **Skills** - Edit `components/Skills.tsx`
5. **Projects** - Edit `components/Projects.tsx`

### Replace Profile Image

1. Add your photo to `public/` folder (e.g., `profile.jpg`)
2. Update `components/ModernHero.tsx`:
   ```tsx
   src="/profile.jpg"  // Change from /placeholder-profile.svg
   ```

### Change Colors

Update the color scheme in `app/globals.css`:
```css
:root {
  --accent: #7bd5ff;      /* Primary accent color */
  --accent-2: #9f7aea;    /* Secondary accent */
  /* Add your colors */
}
```
